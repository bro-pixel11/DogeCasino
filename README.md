🎰 DogeCasino

Cyberpunk-style poker slot machine built with pure HTML, CSS and JavaScript.

⚠️ This project is currently in development.
Some systems, balancing, animations and features are unfinished.

✨ Features

* 🎴 Poker hand detection system
* 💎 Blue card mechanic
* 🎰 Slot machine gameplay
* 🔥 Jackpot animations & effects
* 📊 RTP tracking system
* 💾 Local save system
* 🤖 Autoplay mode
* 📱 Mobile responsive UI
* 🎵 Dynamic sound effects
* 🌌 Cyberpunk neon design

🎮 Game Modes

* Normal Mode (6 cards)
* Turbo Mode (fast spins)
* Madness Mode (7 cards)

🃏 Winning Combinations

* Jacks or Better
* Two Pair
* Three of a Kind
* Straight
* Flush
* Full House
* Four of a Kind
* Straight Flush
* Royal Flush
* Special custom combinations
